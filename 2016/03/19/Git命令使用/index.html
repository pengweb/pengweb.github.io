Git命令使用

1.创建本地仓库

2.发送到远程服务器

3.去除远程子模块

4.远程公共自模块submodule

``` bash
//为当前工程添加submodule
git submodule add 仓库地址 路径
//删除 – 执行下边命令后还要对.gitmodules文件和git中进行删除
git rm –cached
//下载的工程含有submodule，进行更新
git submodule update –init –recursive
